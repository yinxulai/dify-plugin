This plugin is developed and maintained by **Qiniu Cloud** (Shanghai Qiniu Information Technologies Co., Ltd.).

This plugin acts as a pass-through connector between Dify and the Qiniu Cloud Object Storage (Kodo) API. It does **not** independently collect, store, log, or share any user data. File content, bucket metadata, and API credentials are transmitted to Qiniu Cloud solely to fulfill your requests.
