## Data Processed by This Plugin

This plugin acts as a pass-through connector between Dify and the Qiniu Cloud AI inference API. It does **not** independently collect, store, log, or share any user data.

The following data is transmitted to Qiniu Cloud's servers solely to fulfill your requests:

- Prompts and messages you send to the AI model
- Model responses returned to Dify
- API credentials (Access Key / Secret Key) used to authenticate requests
