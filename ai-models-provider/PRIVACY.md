# Privacy Policy

## Data Collection

This plugin does **not** collect, store, or transmit any personal data on its own.

## How This Plugin Works

This plugin routes your prompts and model requests to the [Qiniu Cloud AI inference API](https://openai.qiniu.com/). The data flow is:

1. Your input (prompts, messages) is sent directly from Dify to Qiniu Cloud's API endpoint.
2. The model response is returned to Dify.
3. This plugin does not store, log, or share any of this data independently.

## Third-Party Services

All AI inference is performed by **Qiniu Cloud**. By using this plugin, your data is subject to:

- [Qiniu Cloud Privacy Policy](https://www.qiniu.com/privacy)
- [Qiniu Cloud Terms of Service](https://www.qiniu.com/agreements)

## Credentials

Your Qiniu Cloud API credentials (Access Key / Secret Key) are stored securely within your Dify instance and are only used to authenticate requests to Qiniu Cloud's API.

## Contact

For privacy-related questions, please contact: support@qiniu.com
